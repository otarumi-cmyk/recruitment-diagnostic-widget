## ◆ 吹き出しデザイン詳細

### 吹き出しの形状

```
PC版・タブレット版:
┌─────────────────────────────────┐
│ 「あなたの市場価値での合格率は  │
│   こちらです！」                 │
└─────────────────────────────────┘
         ▲
         │ (しっぽがキャラクターを指す)

スマホ版:
         ┌───────────────────────┐
         │ 「あなたの市場価値での  │
         │   合格率はこちらです！」 │
         └───────────────────────┘
              ▲
              │ (しっぽが上向き)
```

### スタイル仕様

- **背景色**: 白（#ffffff）または薄いオレンジ（#fffaf5）
- **枠線**: 1px solid #e9e5dd（オレンジ系の薄い色）
- **角丸**: 12px
- **影**: 0 4px 12px rgba(0,0,0,0.08)（軽い影）
- **フォントサイズ**: PC 16px、タブレット 15px、スマホ 14px
- **フォントウェイト**: 600（セミボールド）
- **パディング**: 12px 16px
- **しっぽ**: 三角形（CSS borderで実装、幅8px、高さ8px）

### キャラクター画像

- **サイズ**: PC 120px × 120px、タブレット 100px × 100px、スマホ 80px × 80px
- **配置**: 吹き出しの左側（PC・タブレット）、上側（スマホ）
- **角丸**: 50%（円形）
- **影**: 0 2px 8px rgba(0,0,0,0.1)

### 吹き出しのサイズ（キャラクターと統一）

- **高さ**: キャラクターと同じ高さに合わせる
  - PC: 120px（キャラクターと同じ）
  - タブレット: 100px（キャラクターと同じ）
  - スマホ: 80px（キャラクターと同じ）
- **幅**: 自動調整（コンテンツに応じて）
- **パディング**: 上下12px、左右16px（高さを維持するため調整可能）

## ◆ アニメーション仕様

### フェードインアニメーション

```
初期状態:
  - キャラクター: opacity: 0, transform: scale(0.8)
  - 吹き出し: opacity: 0, transform: translateY(-10px)

アニメーション後（0.6秒）:
  - キャラクター: opacity: 1, transform: scale(1)
  - 吹き出し: opacity: 1, transform: translateY(0)

イージング: ease-out
```

### ホバー効果（PCのみ）

```
キャラクター:
  - transform: scale(1.05)
  - transition: 0.2s ease

吹き出し:
  - box-shadow: 0 6px 16px rgba(0,0,0,0.12)
  - transition: 0.2s ease
```

## ◆ レスポンシブ対応

### ブレークポイント

- **PC**: 1080px以上
  - キャラクター: 左側、120px × 120px
  - 吹き出し: 右側、横並び、高さ120px（キャラクターと同じ）

- **タブレット**: 640px〜1080px
  - キャラクター: 左側、100px × 100px
  - 吹き出し: 右側、横並び、高さ100px（キャラクターと同じ）

- **スマホ**: 640px以下
  - キャラクター: 上側、80px × 80px
  - 吹き出し: 下側、縦並び、高さ80px（キャラクターと同じ）
  - しっぽ: 上向き
