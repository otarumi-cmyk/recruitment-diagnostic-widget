# 想定年収診断ツール UIデザイン

## デザインコンセプト

既存の診断ツール（career-probability.html）と同じオレンジテーマを採用し、統一感のあるモダンなUIを実装します。

---

## 1. 診断画面（質問画面）

### レイアウト構成

```
┌─────────────────────────────────────────┐
│  [オレンジグラデーション背景のヘッダー]   │
│  想定年収診断                            │
│  あなたのスキルから想定年収を算出します  │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│  [プログレスバー]                       │
│  ████████░░░░░░░░░░  2/10 (20%)        │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│  [質問カード - ベージュ背景]            │
│                                         │
│  Q1: 現在の年収はどれに該当しますか？  │
│                                         │
│  ┌─────────┐ ┌─────────┐              │
│  │〜300万円 │ │300〜400万│              │
│  └─────────┘ └─────────┘              │
│  ┌─────────┐ ┌─────────┐              │
│  │400〜600万│ │600〜800万│              │
│  └─────────┘ └─────────┘              │
│  ┌─────────┐ ┌─────────┐              │
│  │800〜1000万│ │1000万円以上│            │
│  └─────────┘ └─────────┘              │
│                                         │
│  [戻る]              [次へ]            │
└─────────────────────────────────────────┘
```

### デザイン詳細

#### ヘッダー（hero）
- **背景**: オレンジグラデーション（#ff6b03 → #ffa347）の薄いバージョン
- **テキスト**: 
  - タイトル: 「想定年収診断」（24px、太字）
  - サブタイトル: 「あなたのスキルから想定年収を算出します」（14px、グレー）

#### プログレスバー
- **背景**: 薄いベージュ（#f4ede5）
- **バー**: オレンジグラデーション（#ff6b03 → #ffa347）
- **テキスト**: 「2/10 (20%)」（右寄せ、グレー）

#### 質問カード
- **背景**: 薄いベージュ（#fffaf5）
- **ボーダー**: 薄いベージュ（#e9e5dd）
- **角丸**: 16px
- **影**: ソフトシャドウ

#### 選択肢ボタン
- **通常状態**:
  - 背景: 白（#ffffff）
  - ボーダー: 薄いベージュ（#e9e5dd）
  - 影: 軽いシャドウ
  - ホバー: ボーダーがオレンジに変わり、少し上に浮く

- **選択状態**:
  - 背景: オレンジの薄い色（#fff2e7）
  - ボーダー: オレンジ（#ff6b03）
  - 影: オレンジのシャドウ
  - テキスト: 濃いオレンジ（#7a2f00）

#### ナビゲーションボタン
- **戻るボタン**: 白背景、オレンジボーダー、オレンジテキスト
- **次へボタン**: オレンジグラデーション背景、白テキスト、シャドウ付き

### デザイン詳細

#### 結果表示エリア
- **年収表示**:
  - 大きな数字（48px、太字、オレンジ）
  - 中央配置
  - 背景: オレンジの薄いグラデーション
  - 角丸: 20px
  - 影: ソフトシャドウ

#### 比較表示
- **現年収**: グレーテキスト（14px）
- **加算額**: 緑色テキスト（+125万円、16px、太字）
- **区切り線**: 薄いグレー

#### スキル評価内訳（オプション）
- **カード形式**: 各スキル項目をカードで表示
- **星評価**: ★で視覚的に表示
- **背景**: 薄いベージュ

## 4. レスポンシブデザイン

- 最大幅: 1080px
- 中央配置
- 2カラムグリッド（選択肢）

### タブレット（641px〜1079px）
- 1カラムレイアウト
- フォントサイズ調整

### スマホ（640px以下）
- 1カラムレイアウト
- パディング: 16px
- ボタン: 全幅
- キャラクター画像: 小さめ（80px）
- 吹き出し: 上向きのしっぽ

## 7. 実装イメージ

### 質問画面の特徴
- クリーンで見やすいレイアウト
- 選択肢が明確に区別される
- 進捗が一目でわかる
- スムーズな操作感

### 結果画面の特徴
- 年収が大きく目立つ
- 現年収との比較がわかりやすい
- スキル評価の内訳で納得感を提供
- 再診断が簡単にできる

---

## 8. アクセシビリティ

- コントラスト比: WCAG AA準拠
- キーボード操作: Tabキーで選択肢を移動
- フォーカス表示: オレンジのアウトライン
- スクリーンリーダー対応: 適切なaria-label
