# 想定年収診断ツール 要件定義書

## 1. 概要

### 1.1 目的
ユーザーのスキルと現年収を基に、転職時の想定年収を算出する診断ツールを提供する。

### 1.2 基本方針
- 10個の質問でスキルと現年収を収集
- 現年収を基準に、スキル評価に応じて年収増減を算出
- 基本的には現年収+100万前後を想定年収として表示
- スキルが高い場合は+100万以上、低い場合は+100万未満も可

## 2. 機能要件

### 2.1 質問項目（10問）

#### Q1: 現在の年収
- **目的**: 基準年収を取得
- **選択肢**:
  - 〜300万円
  - 300〜400万円
  - 400〜600万円
  - 600〜800万円
  - 800〜1000万円
  - 1000万円以上

#### Q2: 現在の職種
- **目的**: 職種による市場価値を考慮
- **選択肢**:
  - 一般事務・アシスタント
  - 営業・セールス
  - マーケティング・企画・PdM
  - IT・エンジニア
  - AI・データ・高度専門職
  - その他専門職

#### Q3: 現在の役職・職位
- **目的**: 役職による年収影響を考慮
- **選択肢**:
  - 一般職
  - リーダー・主任
  - マネージャー
  - 部長・ディレクター
  - 事業責任者・役員

#### Q4: プログラミング・デジタルツールの活用経験
- **目的**: デジタルスキルの評価
- **選択肢**:
  - ほぼ使えない
  - Excel・Wordなど一般ツール
  - 業務改善でツールを活用
  - プログラミング・AI活用で成果を出している

#### Q5: プロジェクト管理・推進経験
- **目的**: マネジメントスキルの評価
- **選択肢**:
  - 経験なし
  - 小規模プロジェクト（3名以下）
  - 中規模プロジェクト（5〜10名）
  - 大規模プロジェクト（10名以上）
  - 複数プロジェクトを同時管理
  - 事業レベルのプロジェクトを主導

#### Q6: 過去1年間で達成した具体的な成果
- **目的**: 実績・成果の評価
- **選択肢**:
  - 成果なし
  - 定性的な成果のみ
  - 一部で数値目標を達成
  - 主要な数値目標を達成
  - 数値目標を大幅に達成

#### Q7: 月間の売り上げ（該当する場合）
- **目的**: 営業・売上貢献度の評価
- **選択肢**:
  - なし（売上に関与していない）
  - 30万円未満
  - 30万円〜50万円
  - 50万円〜100万円
  - 100万円〜200万円
  - 200万円〜500万円
  - 500万円〜1000万円
  - 1000万円以上

#### Q8: 担当している業務の範囲
- **目的**: 業務範囲の広さを評価
- **選択肢**:
  - 単一業務のみ
  - 複数業務を担当
  - 部門横断プロジェクトに参加
  - 事業全体に関わる業務

#### Q9: 業務上の意思決定権限
- **目的**: 責任範囲・権限の評価
- **選択肢**:
  - 指示された業務を実行
  - 一部の業務で自律判断
  - 担当領域で意思決定
  - 戦略レベルの意思決定

#### Q10: 保有している資格・スキル
- **目的**: 専門資格・スキルの評価
- **選択肢**:
  - 資格なし
  - TOEIC（700点以上）・簿記2級・基本情報技術者など
  - 税理士・公認会計士・弁護士・医師・薬剤師・中小企業診断士・社会保険労務士・行政書士・宅建士など

## 3. 年収算出ロジック

### 3.1 基本計算式
```
想定年収 = 現年収（中央値） + スキル評価による加算額
```

### 3.2 現年収の中央値マッピング
- 〜300万円 → 250万円
- 300〜400万円 → 350万円
- 400〜600万円 → 500万円
- 600〜800万円 → 700万円
- 800〜1000万円 → 900万円
- 1000万円以上 → 1200万円（上限）

### 3.3 スキル評価スコア

各質問に点数を設定し、合計スコアを計算：

| 質問 | 選択肢 | 点数 |
|------|--------|------|
| Q2: 職種 | 一般事務・アシスタント | 0 |
| | 営業・セールス | 4 |
| | マーケティング・企画・PdM | 6 |
| | IT・エンジニア | 8 |
| | AI・データ・高度専門職 | 10 |
| | その他専門職 | 6 |
| Q3: 役職 | 一般職 | 0 |
| | リーダー・主任 | 3 |
| | マネージャー | 6 |
| | 部長・ディレクター | 8 |
| | 事業責任者・役員 | 10 |
| Q4: デジタルスキル | ほぼ使えない | 0 |
| | Excel・Wordなど一般ツール | 2 |
| | 業務改善でツールを活用 | 4 |
| | プログラミング・AI活用で成果 | 5 |
| Q5: プロジェクト管理 | 経験なし | 0 |
| | 小規模（3名以下） | 2 |
| | 中規模（5〜10名） | 4 |
| | 大規模（10名以上） | 6 |
| | 複数プロジェクト同時管理 | 8 |
| | 事業レベル主導 | 10 |
| Q6: 成果 | 成果なし | 0 |
| | 定性的な成果のみ | 2 |
| | 一部で数値目標達成 | 4 |
| | 主要な数値目標達成 | 6 |
| | 数値目標大幅達成 | 8 |
| Q7: 月間売上 | なし | 0 |
| | 30万円未満 | 1 |
| | 30〜50万円 | 2 |

| | 100〜200万円 | 4 |

| Q8: 業務範囲 | 単一業務のみ | 0 |
| | 複数業務を担当 | 4 |
| | 部門横断プロジェクト | 7 |

| | 戦略レベルの意思決定 | 5 |

| | TOEIC・簿記など | 1 |
| | 高度専門資格 | 2 |

**最大スコア**: 約60点

### 3.4 スコアから年収加算額への変換

スコアを年収加算額に変換するロジック：

```
スコア率 = スコア / 最大スコア(60)
加算額 = 50万円 + (スコア率 × 150万円)
```

**計算例**:
- スコア30点（50%）→ 加算額 = 50 + (0.5 × 150) = 125万円
- スコア45点（75%）→ 加算額 = 50 + (0.75 × 150) = 162.5万円
- スコア60点（100%）→ 加算額 = 50 + (1.0 × 150) = 200万円

**加算額の範囲**: 50万円〜200万円

### 3.5 最終的な想定年収

```
想定年収 = 現年収（中央値） + 加算額
```

**表示形式**: 
- 「想定年収: ○○○万円」
- 「現年収から +○○万円」も併記

## 4. UI/UX要件

## 8. 注意事項

### 8.1 年収算出の精度
- 本ツールは参考値であり、実際の転職時の年収を保証するものではない
- 業界・企業・地域により実際の年収は大きく異なる可能性がある

### 8.2 プライバシー
- 年収情報は機密情報のため、適切に取り扱う
- サーバーに送信しない（クライアントサイドのみで処理）
